# Tabelas Verdade dos Operadores Lógicos em Programação

## Operador AND (&&)

| A     | B     | A && B |
|-------|-------|--------|
| true  | true  | true   |
| true  | false | false  |
| false | true  | false  |
| false | false | false  |

---

## Operador OR (||)

| A     | B     | A \|\| B |
|-------|-------|----------|
| true  | true  | true     |
| true  | false | true     |
| false | true  | true     |
| false | false | false    |

---

## Operador NOT (!)

| A     | !A    |
|-------|-------|
| true  | false |
| false | true  |
